Design preferences for the dealpulseksa.com website:

- **Visual style**: Apple-inspired (clean, premium, glassmorphism, smooth animations, generous whitespace).
- **Background**: Logo as a large, faded, watercolor-style watermark fixed behind ALL content. Light cream/white base color.
- **Structural reference**: arabiccoupon.com (sections, navigation, store grid layout) — but visual aesthetic is Apple, not the orange-heavy arabiccoupon look.
- **Brand colors**: Deal Pulse green #10B981 (accent), Saudi flag green for highlights, white/cream base.
- **Auth**: Firebase Auth with phone OTP — chosen for ease of setup and free tier (10K SMS/month).
- **Build pace**: User wants the full version built in one go, not piecemeal.

**Why:** The user is building a premium Saudi coupon site that needs to feel trustworthy and modern. Apple aesthetic conveys quality; the watermark logo gives unique brand identity differentiating from competitors like arabiccoupon.com.

**How to apply:** Default to light mode with dark toggle. Use blur/glass effects on cards. Logo watermark must be large but subtle (~5-10% opacity) so content remains readable. Phone OTP login is mandatory — no email/password.
